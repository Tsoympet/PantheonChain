# Tokenomics

- **TALANTON (L1):** PoW issuance + L1 fees.
- **DRACHMA (L2):** PoS issuance + L2 tx fees + staking.
- **OBOLOS (L3):** PoS issuance + gas fees + staking.

Parameters are network-profile specific in genesis/config files.
See:

- `genesis_talanton.json`
- `genesis_drachma.json`
- `genesis_obolos.json`
