# Glossary

- **Epoch**: validator scheduling interval in PoS layers.
- **Finality**: >=2/3 stake signatures confirming a block/commitment.
- **Commitment**: summary hash package anchored to lower layer.
- **Settlement**: irreversible accounting at L1.
- **Relayer**: service that submits higher-layer commitments to lower layer.
- **Devnet**: local developer network profile.
