# Security Policy

## Reporting vulnerabilities

Please report vulnerabilities privately to maintainers before public disclosure.

- Preferred: GitHub Security Advisories for this repository.
- Include: impact, affected components, reproduction steps, and suggested remediation.

## Scope

- Node daemon (`clients/core-daemon`)
- RPC and CLI surfaces
- Relayers and configuration tooling

Do not publish zero-day details publicly until a fix and coordinated disclosure timeline is agreed.
