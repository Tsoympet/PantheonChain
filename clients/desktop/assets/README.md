# Desktop App Assets

This directory contains the visual assets for the ParthenonChain desktop wallet application.

## Platform App Icons

- `icon.png` (512x512)
- `icon.ico` (Windows multi-size icon)
- `icon.icns` (macOS app icon bundle)

## UI SVG Icons

The following SVG icons are available for desktop UI/navigation usage:

- `home.svg`
- `send.svg`
- `receive.svg`
- `transactions.svg`
- `menu.svg`
- `wallet.svg`
- `mining.svg`
- `settings.svg`

## Usage

These assets can be referenced directly in Qt resources or documentation examples.
