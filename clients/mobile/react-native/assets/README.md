# Mobile App Assets

This directory contains the visual assets for the ParthenonChain mobile wallet application.

## Existing App Assets

- `icon.png` (1024x1024) - Main app icon
- `splash.png` (2048x2048) - Splash screen image
- `adaptive-icon.png` (1024x1024) - Android adaptive icon foreground
- `favicon.png` (32x32) - Web favicon
- `icon.svg`, `logo.svg`, `favicon.svg` - source SVG files

## Navigation/Feature SVG Icons

Added reusable SVG icons for core wallet features:

- `home.svg`
- `send.svg`
- `receive.svg`
- `transactions.svg`
- `menu.svg`
- `wallet.svg`
- `mining.svg`
- `settings.svg`

## Usage

These icons are intended for use in React Native navigation tabs, buttons, and feature headers.
