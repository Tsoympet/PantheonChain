# ParthenonChain Assets

This directory contains the official branding and visual assets for ParthenonChain, designed following strict ancient Greek geometric principles.

## Design Philosophy

All visual assets follow a **geometric minimalist** approach inspired by:
- Ancient Greek architectural proportions  
- Classical temple structures (Parthenon)
- Perfect geometric primitives (circles, squares, triangles, straight lines)
- Mathematical symmetry and order

### Core Principles

1. **Geometric Purity** - Only perfect shapes: circles, squares, triangles, straight lines
2. **Flat Colors** - No gradients, max 2 colors per icon
3. **Symmetry** - Bilateral or radial symmetry throughout
4. **No Decorative Elements** - Functional geometry only
5. **Timeless Aesthetic** - Institutional, mathematical, inevitable

### Color Palette

**Primary Colors:**
- Aegean Blue: `#1F3A5F` (trust, security, protocol)
- Bronze Gold: `#B08D57` (value, classical architecture)

**Secondary Colors (used sparingly):**
- Marble White: `#F5F5F2`
- Obsidian Black: `#0F0F12`
- Oxide Red: `#7A2E2E`
- Basalt Gray: `#2A2A2E`

## Technical Specifications

- Pure SVG 1.1 / SVG 2.0
- No embedded raster images
- No external fonts
- Flat colors only (no gradients)
- Scalable from 16×16 to 1024×1024
